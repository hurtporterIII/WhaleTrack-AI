# WhaleTrack-AI Project Plan

## Objective
Build a real-time automated whale-transaction alert system using:
- Airtable (storage)
- n8n (automation engine)
- Discord (alert delivery)
- Qubic blockchain (data source)

## Deliverables
- Airtable schema (JSON export)
- n8n workflow (JSON export)
- Setup guide
- Demo assets
- Documentation

## Timeline
**Day 1–2:** Build Airtable schema  
**Day 2–3:** Build n8n workflow  
**Day 3–4:** Documentation + testing  
**Day 5:** Record demos, finalize submission

## Evaluation Focus
- Clear automation pipeline  
- Clean repo structure  
- Easy reproducibility  
- Practical value for Qubic ecosystem
